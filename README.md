# catDB

## the REST API for catDB is as follows
1. GET requests
  * /randomVideo
  * /getAllVideo
  * /getPopularVideos
  * /getVideoByUser/{userId}
  * /getVideoByVidId/{catVidId}
  * /getVideoByTag
  * /getComments/{catVidId
  * /getAward
  * /getTag
  * /getTags/{catVidId}
2. PUT requests
  * /upMeow/{catVidId}
  * /downMeow/{catVidId}
  * /addAward/{catVidId}/{awardId}
3. POST requests
  * /addVideo
  * /postComment
